# RF
[X] O usuário deve poder criar uma nova transação;
[] O usuário deverá poder ter um resumo da sua conta;
[X] O usuário deverá poder listar todas as transações que já ocorreram;
[X] O usuário deverá poder visualizar uma transação única;

# RN
[X] A transação poderá ser do tipo crédito que somará ao valor total ou débito que subtrairá do valor total;
[] Deve ser possível identificar o usuário entre as requisições;
[] O usuário só pode visualizar transações o qual ele criou;
